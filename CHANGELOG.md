# Changelog

- Initial public release of ComicShield.
- Added click-trap protection for observed invisible empty-space popup surfaces.

- Added floating reader controller.
- Added requestAnimationFrame autoscroll.
- Added reader zoom controls.
- Added official ComicShield shield/comic icon assets.
