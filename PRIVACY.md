# Privacy Policy

ComicShield is designed to run locally in Chrome on comix.to pages.

## Data Collection

ComicShield does not collect, transmit, sell, or share user data. It does not use analytics, telemetry, tracking pixels, remote logging, or external servers.

## Accounts

ComicShield does not collect account identifiers, usernames, passwords, tokens, or payment information.

## Network Requests

The extension does not fetch remote scripts or send data to external services. It runs as local extension code on the supported host permissions declared in `manifest.json`.

## Local Storage

ComicShield stores settings locally through `chrome.storage.local`, including toggles, reader controller state, autoscroll speed, and reader zoom preferences.

## Host Scope

The extension is scoped to:

- `https://comix.to/*`
- `https://*.comix.to/*`

No other website host permissions are requested.
