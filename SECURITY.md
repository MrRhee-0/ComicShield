# Security Policy

## Reporting Security Issues

If you believe you found a security issue in ComicShield, please report it privately to the project maintainer instead of posting sensitive exploit details in a public issue. If private contact is not available, open a GitHub issue with a minimal description and omit exploit steps until a maintainer responds.

Please include:

- ComicShield version.
- Chrome version.
- The comix.to URL path involved, if safe to share.
- A concise description of the behavior.
- Screenshots only when they do not expose private information.

## Scope

ComicShield is scoped to extension behavior and supported comix.to pages. It is not antivirus software, malware removal software, or a guarantee of total protection. It neutralizes observed intrusive click-trap and overlay surfaces without opening suspicious links.

## Supported Site Scope

This release supports comix.to and relevant subdomains through the host permissions in `manifest.json`. Other sites require separate adapters and review.
